# whmcs-chinese
一个适用于whmcs 9.04的中文语言包

在whmcs 9.x的更新中，whmcs改动了很多字段，导致使用旧的中文汉化文件的情况下打开部分页面会出现“Cannot access offset of type string on string in”的报错

加上某些旧的中文语言包普遍存在雷石东直放站和翻译不全的问题，本汉化文件使用 gpt 5.5完成翻译工作，并且没有强行翻译某些专有名词

使用方式

将chinese.php放入
```
/admin/lang
```

然后在管理员中选择当前登录的管理员，点击编辑，语言选择 chinese即可
